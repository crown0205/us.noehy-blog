# 면접 질문

## 1. 자기소개

- 자기소개를 해주세요.

## 2. 프로젝트

- 프로젝트를 하나 소개해주세요.

## 3. 프로젝트 질문

- 프로젝트를 하면서 어려웠던 점이 있나요?

- 프로젝트를 하면서 가장 자신있는 부분이 무엇인가요?

- 프로젝트를 하면서 사용한 기술 스택은 무엇인가요?

- 프로젝트를 하면서 사용한 기술 중 익숙한 기술은 무엇인가요?

- 프로젝트를 하면서 사용한 기술 중 익숙하지 않은 기술은 무엇인가요?

- 프로젝트를 하면서 사용한 기술 중 공부하면서 가장 어려웠던 기술은 무엇인가요?

## 4. 기술 질문

- 리엑트 렌더링 방식에 대해 설명해주세요.

- 렌더링 최적화 방법에 대해 설명해주세요.

- 부모 페이지에서 section이 세개가 있다. 각 section은 overflow는 scroll이 된다. 
  scroll 하다가 부모 페이지에서 렌더링이 일어날때, scroll의 위치도 초기화가 되는데 이를 방지하는 방법은 무엇이 있을까요?

    ㄴ scroll 위치를 저장하는 방법이 있습니다. scroll 위치를 저장하는 방법을 사용하여 scroll 위치를 초기화하지 않도록 할 수 있습니다.

- atomic design에 대해 설명해주세요.

  - 사용하면서 단점이 무엇이라고 생각하나요?

    ㄴ atomic design은 재사용성이 높아지고, 유지보수가 쉬워지는 장점이 있지만, 컴포넌트의 수가 많아지면 관리하기 힘들다는 단점이 있습니다.
    ㄴ 또한, atomic design을 사용하면서 컴포넌트의 수가 많아지면서, 컴포넌트의 이름을 지을 때 어려움을 겪을 수 있습니다.
    ㄴ 상태 관리를 하기 위해 context api나 redux를 사용할 때, atomic design을 사용하면서 상태를 관리하기 어려울 수 있습니다.
 
```jsx
// atomic design 예시

  return (
    <div>

  );

export default Atom;
```
