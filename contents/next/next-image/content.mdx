# Next image 사용법

Next.js에서 이미지 사용법을 알아보자.

## 이미지 추가하기

이미지를 추가하는 방법은 다음과 같다.

```jsx
import Image from 'next/image'

function MyImage() {
  return (
    <Image
      src="/images/profile.jpg"
      alt="Picture of the author"
      width={500}
      height={500}
    />
  )
}

export default MyImage
```
